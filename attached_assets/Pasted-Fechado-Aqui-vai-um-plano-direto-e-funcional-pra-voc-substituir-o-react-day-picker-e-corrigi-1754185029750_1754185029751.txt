## ✅ **Plano de Substituição + Correção de Bugs no Replit**

### 🔁 **1. Substituir `react-day-picker` por `react-datepicker`**

**Motivo:** Compatível com `React 19`, leve, estável, estilizável com Tailwind.

#### A. Remover o antigo

```bash
npm uninstall react-day-picker
```

#### B. Instalar o novo

```bash
npm install react-datepicker --legacy-peer-deps
```

#### C. Importar no seu componente

```tsx
import DatePicker from "react-datepicker";
import "react-datepicker/dist/react-datepicker.css";
```

#### D. Exemplo básico de uso (pra testar)

```tsx
const [startDate, setStartDate] = useState(new Date());
<DatePicker selected={startDate} onChange={(date) => setStartDate(date)} />
```

### 🧩 **2. Resolver conflito do `tailwindcss-animate`**

#### A. Instalar o plugin corretamente

```bash
npm install tailwindcss-animate --legacy-peer-deps
```

#### B. Adicionar no `tailwind.config.ts`

```ts
plugins: [require("tailwindcss-animate")],
```
